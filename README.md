#  Bouncing Ball (Three.js)

A simple, lightweight 3D simulation of a bouncing ball built using **Three.js** and Vite. 

##  Features
* Smooth 3D animation of a bouncing ball.
* Clean, minimalist setup.
* Fast local development with Vite.

##  How to Run Locally

Follow these quick steps to get the project running on your computer:

### 1. Clone the repository
```bash
git clone [https://github.com/yasserotani/Bouncing-Ball---Three-js.git](https://github.com/yasserotani/Bouncing-Ball---Three-js.git)
cd Bouncing-Ball---Three-js
```
### 2. Install dependencies
Make sure you have Node.js installed, then run:
```bash
npm install
```
### 3. Start the development server
```
npm run dev
```
After running this command, click the local link provided in your terminal (usually http://localhost:5173) to view the bouncing ball in your browser!

Tech Stack
Three.js - For creating the 3D graphics.
Vite - For the fast frontend tooling and development server.
